18. What is SVG?

ANS :- SVG stands for Scalable Vector Graphics. It is a widely used XML-based file format for describing two-dimensional vector graphics.
Key features of SVG include:
1.Scalability: SVG images can be scaled to any size without losing quality. This makes them suitable for a variety of applications, from small icons to large graphics.
2.XML Format: SVG files are written in XML (eXtensible Markup Language), a text-based format that is both human-readable and machine-readable. This makes SVG files easy to create, modify, and manipulate.
3.Interactivity: SVG supports interactivity and animation. You can use JavaScript to add interactive elements to SVG graphics, making them dynamic and engaging.
4.Accessibility: SVG supports accessibility features, making it possible to create graphics that are friendly to users with disabilities. This is particularly important for web design and other applications where accessibility is a priority.
5. Text-Based: Being text-based, SVG files can be compressed, making them smaller in file size. This can lead to faster loading times for web pages and reduced bandwidth usage.
6.Wide Browser Support: Most modern web browsers support SVG, making it a popular choice for web design and development.
